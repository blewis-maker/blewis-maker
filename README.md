### Brandan Lewis

I build **[Agix](https://agix-ai.io)** — and design the agentic ecosystems that run on it.

My focus is **agentic ecosystem architecture**: designing how teams of AI agents coordinate, remember, govern their own actions, and improve over time — so a business can run as a coordinated *fleet* of agents instead of a pile of disconnected tools.

Agix is that work as a product, on two surfaces:

- **Agix AOS** — the open-source core. An operating system for a governed, legible fleet of AI agents you own and run on your own machine. *(Open-sourcing now.)*
- **Agix** — agentic implementation work for teams that want the foundation designed and built with them.

The goal is simple: give people an AI **foundation they own and grow** — not features they rent — and build it so it gets more capable over time instead of more tangled.
